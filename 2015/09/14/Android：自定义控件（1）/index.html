一、创建一个View

public class CustomView extends View{
    // 必要添加的两个构造器
    public CustomView(Context context){
        super(context);
    }

    }
}

二、Android的View绘制流程

Android的View绘制流程

自定义View使用View提供的onDraw(Canvas canvas)方法来绘制自己View。

@Override
protected void onDraw(Canvas canvas){
    super.onDraw(canvas);
}

Canvas可以理解为画布，用来承载View。画布有了，那么笔呢，Android中Paint对象就是我们要的画笔了。

public class CustomView extends View {
    private Paint piant;

    public CustomView(Context context) {
        this(context , null);
    }

    public CustomView(Context context, AttributeSet attrs) {
        super(context, attrs);
        // 不要在onDraw()方法中初始化画笔
        // 因为onDraw()方法是频繁调用的
        initPaint();
    }
    private void initPaint() {
        // 开启抗锯齿
        Paint paint = new Paint(Paint.ANTI_ALIAS_FLAG);
        // or Paint paint = new Paint();
        // paint.setAntiAlias(true); 同样解决抗锯齿的问题
    }

    @Override
    protected void onDraw(Canvas canvas) {
        super.onDraw(canvas);
    }
}

Paint集成所有画笔的属性，Canvas定义画布属性，drawxxx()方法来绘制不同View。如常用的画圆：drawCircle()、画圆弧drawArc()、绘制位图drawBitmap();
画圆：

public class CustomView extends View {
    private Paint paint;
    private int width , height;
    private int top;
    private static String TAG = CustomView.class.getName();
        public CustomView(Context context) {
        this(context, null);
    }

    public CustomView(Context context, AttributeSet attrs) {
        super(context, attrs);
        initPaint();
        initSize();
    }

    private void initSize() {
        DisplayMetrics dm = new DisplayMetrics();
        if(getContext() instanceof Activity){
            ((Activity)getContext()).getWindowManager().getDefaultDisplay().getMetrics(dm);
        }
        width = dm.widthPixels;
        height = dm.heightPixels;
        Log.d(TAG , "width = " + width + " , height = "+ height);
    }

    private void initPaint() {
        // 开启抗锯齿
        paint = new Paint(Paint.ANTI_ALIAS_FLAG);
        // or Paint paint = new Paint();
        // paint.setAntiAlias(true); 同样解决抗锯齿的问题
        // 画笔颜色
        paint.setColor(Color.GRAY);
        /**
        * 画笔三种Style
        * STROKE ：描边
        * FILL_AND_STROKE : 描边并填充
        * FILL ：填充
        */
        paint.setStyle(Paint.Style.STROKE);
        // 画笔宽度
        paint.setStrokeWidth(10);
    }

    @Override
    protected void onDraw(Canvas canvas) {
        super.onDraw(canvas);
        canvas.drawCircle(width / 2, height / 2, 200, paint);
    }
}

ColorFilter 三个直接子类 ColorMatrixColorFilter、LightingColorFilter、PoterDuffColorFilter

1.ColorMarixColorFilter

ColorMatrix colorMatrix = new ColorMatrix(new Float[]{
        1, 0, 0, 0, 0,   // R
        0, 1, 0, 0, 0,   // G
        0, 0, 1, 0, 0,   // B
        0, 0, 0, 1, 0    // A
});

范围0.0F - 2.0F, 1为保持原图RGB值，每一行的第五列表示偏移量。色彩偏移可以基于白平衡来理解。我们定义的“白色”（比如：255, 253, 251, 247）和纯白（255, 255, 255, 255）之间的偏移值（0, 2, 4, 8);
2.LightingColorFilter

LightingColorFilter(int mul , int add);

mul 全称colorMultiply：色彩倍增，add全称 colorAdd色彩添加，两个值均为16进制的色值0xAARRGGBB。
当LightingColorFilter(0xffffffff , 0x00000000)的时候原图不会有任何变化。增加红色使用LightingColorFilter(0xffffffff,0x00xx0000)就好，其中xx取值为00-ff。这个方法可以用在点击图片变色的功能上，而无需多准备另一张点击效果的图片。

setOnClickListerner(new OnClickListener(){

    @Override
    public void onClick(View v){
        if(isClick){
            mPaint.setColorFilter(null);
            isClick = false;
        }else{
            mPaint.setColorFilter(new LightingColorFilter(0xffffffff , 0x00ffff00));
            isClick = true;
        }
        // 
        invalidate();
    }
});

运行后点击图片可变成黄色再次点击变回原色。
